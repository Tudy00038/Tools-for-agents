Agent Name: GitHub README Summarizer
Capabilities: Specialized in fetching and summarizing README.md files from GitHub repositories.
Trigger Conditions: User explicitly mentions GitHub URLs, asks about repository details, or requests summaries of READMEs.

Agent Name: Search Web Agent
Capabilities: Searches the web for answers to general questions and translates text between languages.
Trigger Conditions: When a user asks general information questions or explicitly requests translation between languages.

Agent Name: Code Generator Agent
Capabilities: Can generate code in Python from user input and can execute it if needed.
Trigger Conditions: When a user requests a simple algorithm or asks how to write a function in Python.
